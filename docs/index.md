# Ssc-gen

## Install

ssc_gen required python 3.10 version or higher

| package manager | command                       |
|-----------------|-------------------------------|
| pipx            | `pipx install ssc_codegen`    |
| pip             | `pip install ssc_codegen`     |
| uv              | `uv tool install ssc_codegen` |
